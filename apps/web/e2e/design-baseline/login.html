AC Agent 控制台

登录控制台

私有访问边界，登录即获得 host-root 执行权限。平台不开放注册，账号由管理员开通。

忘记密码？切换到「邮箱验证码」登录，或联系管理员重置。

验证码 10 分钟内有效，60 秒内可重发一次；仅向已开通的邮箱发送。

仅 AUTH_ALLOWLIST 白名单内的 GitHub 账号可进入；从任务会话被拦截登录会回到原页面。

没有账号？平台不开放公开注册——请联系管理员在「设置 · 账号」中为你开通私有账号。
